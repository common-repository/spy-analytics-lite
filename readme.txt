You can record and watch "video" what visitors did on your page: mouse movements, clicks and scroll

== Description == 

**Features:**

1. Record and view mouse move actions;
2. Record and view window resize actions;
3. Record and view page scroll actions;
4. Record and view mouse clicks (FREE to unlock!);
5. Detect user IP, Country, OS, Browser, Mobile, Wp user name (FREE to unlock!);

== Installation ==

### How to install ###
1. Download the ZIP package.
2. Open WordPress admin and go to Plugins -> Add New -> Upload. Browse for the ZIP file spy-analytics-lite.zip on your computer and hit “Install Now”.
3. Activate the plugin.

### Quick Start Quide ###

1. Install Spy Analytics Lite
2. Open Spy Analytics Lite - Settings. Check Enable record checkbox.  Save settings
3. After a while, open Spy Analytics Lite - Analytics page. If you have recorded session you will see it in the table. If there are no sessions you can record your own, just open any page of your website and make mouse/window resize manipulations
4. Choose one of recorded session in the sessions table. Put mouse cursor over Session ID value to open session menu. Click Spy View link. Spy Player page will opens in new tab/page.
5. Click Play button to view recorded actions.

== Frequently asked questions ==

= How much memory i need for 1min activity per 1 user? =
It will be about 60 Kb of hyperactivity. But please note that it depends on how it will be active on your website. For example one visitor can click 12 times per 1 min, another one just 3.
